# Storybook React Library
Storybook with react components to create a new library

## :dart: Features

 - Atom components
 - Molecules components
 - Organisms components

## :computer: Tech Stack

 - [Storybook](https://storybook.js.org/)
 - [ReactJS](https://react.dev/)
 - [Typescript](https://www.typescriptlang.org/)

## :gear: Installation
``` bash
# Install dev-dependencies and dependencies
yarn install
```

## :eyes: How to usage

### Start Storybook
``` bash
# Use this command to start storybook in localhost for watch components
yarn storybook
```
### Build storybook
``` bash
# Use this command to build a new static content with storybook
yarn build-storybook
```

### Pre Build library
``` bash
# Use this command to pre build fonts and icons
yarn prebuild-library
```

### Build library
``` bash
# Use this command to build a new library
yarn build-library
```
